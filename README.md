# Gunasekaran S Portfolio

A single-page portfolio website built with plain HTML and CSS.

## Files

- `index.html` — main page markup
- `styles.css` — project styling and layout

## Overview

This portfolio showcases professional projects, experience, education, and contact details for a Lead Robotics Engineer.

## Usage

1. Open `index.html` in a browser.
2. Or serve the directory with a static server if needed.

## Notes

- Uses Google Fonts for typography.
- The site is responsive and designed as a simple landing page for a robotics engineering portfolio.
